# Slideshow System v1

## Objective

Produce TikTok slideshow posts that move viewers from private recognition to app action.

## Winning pattern

```text
private behavior hook
→ hidden loop
→ emotional cost
→ tiny actionable reframe
→ Cherry as structure
→ comment/save/share CTA
→ Google Play CTA
```

## Why previous posts capped around ~800

Hypotheses:

1. Copy was decent but not private-behavior specific enough.
2. Pixel art was visually consistent but not narratively evocative.
3. Posts likely lacked enough comment triggers.
4. App insertion may have arrived too early or too vaguely.
5. Hooks may have felt like general anxiety content rather than “this is exactly me.”

## 8k target levers

1. **Private behavior specificity**: name behaviors people hide.
2. **Prediction violation**: say something counter to generic advice.
3. **Retention bridge**: every slide must create a reason to swipe.
4. **Confession CTA**: invite comments with a low-friction admission.
5. **Saveable tool**: include a mini framework users can reuse.
6. **Late product insertion**: earn attention before product CTA.
7. **Symbolic image progression**: background should feel like a tiny emotional story.

## Visual doctrine

Keep pixelated backgrounds, but make them more evocative by assigning each post a symbolic object that changes across slides:

- blinking cursor → doorway
- receipt printer → field notebook
- unopened envelope → bridge
- empty chair → path
- storm radar → streetlight
- locked room → staircase

No readable text in images. No literal crying faces. No clinical/medical imagery.
